# Focus - Issue Tracker

Welcome! 👋

This repository is the **public issue tracker** for **Focus**, a private, local-first video manager for organizing, searching, transcribing, and managing your video library directly on your computer.

## 🎯 What is this repo for?

This repository is dedicated to:
- 🐛 **Bug Reports** - Report issues you encounter while using Focus
- ✨ **Feature Requests** - Suggest new features or improvements

**Please note:** This repository does **not** contain the source code for Focus. It exists solely for collecting feedback and tracking issues.

## 📝 How to report an issue

1. **Search first** - Check if someone has already reported your issue or requested your feature
2. **Create a new issue** - Click the "Issues" tab above, then "New Issue"
3. **Choose a template** - Select either "Bug Report" or "Feature Request"
4. **Fill in the details** - The more information you provide, the better we can help!

## 🤝 Contributing

We appreciate your feedback! Please read our [CONTRIBUTING.md](CONTRIBUTING.md) guide to learn how you can help make Focus better.

If you have general questions about Focus or need help using the app, please:
- Visit [use-focus.com](https://use-focus.com) for more information
- Look through existing issues to see if your question has been answered
